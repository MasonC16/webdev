# Assignment 2 — Product Manager (Student Starter — STRICT)

This starter compiles and renders a shell only. All assessed logic is left blank.

## Your Tasks (all REQUIRED)
- Build controlled inputs and inline validation for: name, price, stock, description.
- Implement create, edit, delete product flows.
- Persist with localStorage using try/catch; add a Reset Storage button.
- Add search (by name) and sort (name/price, asc/desc) in a toolbar.
- Show confirmation banners for save/delete that auto-dismiss (~2s).
- Render an empty state when no products exist.

## Run

```bash
npm install
npm run dev
```
