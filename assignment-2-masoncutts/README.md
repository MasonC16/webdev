# Overview
 this application is a simple product management tool that allows users to add, view, and manage products. It provides a user-friendly interface for managing product information and inventory. this appllication also allows users to filter products
 based on name or price, in both ascending and descending order. The application also has localstorage implementation, allowing
 users to save the products they've added after refreshing the page, and has a button to remove all products from localstorage.

# Implemented features
- Add products with name, price, quantity, and description
- View a list of products with their details
- Filter products by name or price in ascending and descending order
- Save products to localstorage to persist data across page refreshes
- Remove all products from localstorage with a single button click
- Search for products by name or price

# Run instructions
1. Clone the repository
2. Navigate to the project directory
3. Install dependencies using `npm install`
4. Start the application using `npm start`
5. Open your browser and go to `http://localhost:5173` to view the application
