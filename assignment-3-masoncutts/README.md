# Assignment 3 — Product Catalog

# Overview
This application is a single-page React product catalog designed to demonstrate client-side routing, state management, and
component-based architecture. It provides a structured interface for working with product data and showcases the use of reusable 
components, a custom React hook for managing state, and integration with browser storage. The project emphasizes clean 
organization, user interaction, and dynamic rendering within a modern React application.

# Implemented Features
- Add, edit, and delete products with name, price, stock, category, and description
- View a list of products and detailed product pages
- Search products by name
- Filter products by category and sort by name or price
- Save products to localStorage to persist data across page refreshes
- Navigate between pages using client-side routing with a fallback 404 page

# Run Instructions
1. Clone the repository or download and unzip the folder
2. Navigate to the project directory
3. Install dependencies using `npm install`
4. Start the application using `npm run dev`
5. Open your browser and go to `http://localhost:5173` to view the application
